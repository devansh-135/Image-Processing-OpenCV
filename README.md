# Image Processing Project using Python

## Description
This project demonstrates basic image processing techniques using Python and OpenCV.

## Features
- Grayscale conversion
- Gaussian Blur (noise reduction)
- Edge detection (Canny algorithm)

## Technologies Used
- Python
- OpenCV

## How to Run
1. Install OpenCV: pip install opencv-python
2. Place an image named input.jpg
3. Run: python main.py

## Output
The program generates processed images in the output folder.
